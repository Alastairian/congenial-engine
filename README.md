# congenial-engine

This project is ready for deployment on Vercel.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Local Development:
   ```bash
   npm start
   ```

3. Deploy to Vercel:
   - Push your code to GitHub.
   - Import your repo on [Vercel](https://vercel.com/import).
   - Vercel will detect your settings automatically.
